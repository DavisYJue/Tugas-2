"Tentu saja, TIDAK BISA. Kenapa? Ya karena store nya belum dibuka, sayang. Boro-boro disalurkan ke akun game, Admin mau top-up aja ngutang.

4. Kenapa harga-harga item di PoteStore asal-asalan dan lebih mahal dari store lain ?

Karena Admin hanya menetapkan harga-harga item di PoteStore sesuai dengan FEELING admin dikarenakan Admin tidak pasti akan keberlangsungan store ini.

Untuk sementara waktu, tidak bisa melakukan pembayaran dikarenakan belum ada kerjasama dengan mitra media pembayaran uang digital. Jikalau ingin membayar tanpa mengharapkan imbalan in-game-item, silahkan klik link di kolom jawaban pertanyaan ke-3 untuk berdonasi.
